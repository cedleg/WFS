# WFS

Web Front Seed, minimal stack for JavaScript Project

## Installation

Clone the repo
```
git clone https://github.com/cedleg/WFS.git
```
Install package
```
npm install
```
## Serve
Run the application
```
npm run global
```
## Test
Pass the tests
```
npm run test
```
Generate report
```
npm run coverage
```
